# Hi there, I'm [Muhammad Rivki](https://rivki.dev) — @mikqi 👋

**Senior Frontend Engineer** focused on building fast, scalable, and accessible products.

I currently work at **Govtech Edu Indonesia**, where I help improve digital experiences that support education at scale. My sweet spot is combining strong frontend architecture with better developer experience, so teams can ship confidently and move faster.

## What I focus on

- Designing **maintainable frontend systems** that scale with product and team growth
- Improving **developer experience (DX)** through better workflows, tooling, and standards
- Delivering **high-performance web apps** with strong usability and responsiveness
- Building **accessible interfaces** that work well for everyone

## Core toolkit

- **Frontend:** JavaScript, TypeScript, React, Vue, Next.js, Nuxt, CSS Modules, Tailwind CSS
- **Backend & data:** Node.js, Express.js, GraphQL, MongoDB, MySQL
- **Quality & delivery:** Playwright, Vitest, Jest, Storybook, CI/CD, Docker
